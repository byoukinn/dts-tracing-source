# 适用于 dts 的溯源懒人脚本

使用方法：

1. 按`F12`打开浏览器的开发者控制台
2. 将压缩后的代码粘贴进去敲回车
3. 打开dts的源表节点，出现弹出框后，按alt+x（或者手动输入`auto()`)
4. 网速差的耐心等待跳转到字段页面

> 额外的： 在执行一次脚本后如果觉得内容
